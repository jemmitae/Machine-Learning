Existen hasta el momento cuatro tipos de aprendizaje en machine learning, aprendizaje supervisado, aprendizaje no supervisado, aprendizaje semisupervisado y aprendizaje por refuerzo, en el aprendizaje supervisado se parte de un conocimiento a priori, se ejecuta para deducir una función que haga de la mejor manera posible el mapeo de unos datos de entrada y de salida, busca predecir basado en algunos datos una variable de salida la cual puede ser cuantitativa o cualitativa, toma algunos datos de entrenamiento para generar esta función que minimiza el riesgo empírico sobre el conjunto de entrenamiento. Generalmente para problemas de regresión o problemas de clasificación.
El aprendizaje no supervisado, tiene como objetivo modelar la estructura o distribución de los datos para aprender más sobre ellos, este aprendizaje tiende a ser subjetivo debido a que no tiene respuestas correctas, pueden ser agrupados en algoritmos de clustering y algoritmos de asociación.
El aprendizaje semisupervisado, en este, se tienen los dos tipos de datos anteriores combinados, para ello este tipo de aprendizaje busca encontrar y construir un modelo supervisado con ellos que sea el mejor. Generalmente quienes usan este aprendizaje corresponden self-training, co-training, assemble y Re-Weighting.
El ultimo tipo es el aprendizaje por refuerzo, acá se tienen y se manejan problemas no son supervisados que solo reciben re-alimentaciones o refuerzos, se sustituye la información supervisada y se reemplaza por información del tipo acción/reacción, su objetivo es aprender a mapear situaciones de acciones para maximizar una función de recompensa. En cada interacción el agente recibe como entrada un indicador de estado actual y selecciona una determinada acción que maximice una función de refuerzo o recompensa a largo plazo. Este tipo de proceso se puede caracterizar como un proceso de Markov.
Este proceso es un fenómeno aleatorio  dependiente del tiempo para el cual se cumple una propiedad específica.
